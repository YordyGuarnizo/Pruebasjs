Ejemplos de Eventos y Selección de Elementos

1. Evento 'click' y getElementById()

Al hacer clic en el botón, el código JavaScript cambiará el texto de abajo.

Texto original (Aún no has hecho clic)

2. Evento 'input' y querySelector()

El evento input se dispara en tiempo real cada vez que escribes, borras o modificas el contenido.

3. Evento 'change'

El evento change se dispara cuando confirmas un cambio. En un select se nota de inmediato al elegir otra opción.

4. Evento 'submit'

El evento submit se detecta en el formulario. Usaremos preventDefault() para evitar que la página se recargue.

5. Eventos 'mouseover' y 'mouseout' (Ratón)

Se disparan cuando el puntero del ratón entra (mouseover) y sale (mouseout) de un elemento.

Pasa el ratón por aquí

6. Eventos de Teclado ('keydown' y 'keyup')

keydown: se dispara al presionar una tecla hacia abajo.
keyup: se dispara al soltar la tecla.

7. Eventos 'focus' y 'blur'

focus: ocurre cuando un elemento (como un input) recibe la "atención" (cuando haces clic para escribir en él).
blur: ocurre cuando el elemento pierde la atención (haces clic fuera).
